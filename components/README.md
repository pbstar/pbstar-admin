# PComponents 组件库

## 快速开始

### 引入

```javaScript
// 全局引入并注册
import pComponents from '../../components';
app.use(pComponents);
// 页面按需引入
import pVerificationCode from '@PComponents/verificationCode';
```

### 使用组件

```javaScript
<template>
  <div>
    <p-button>按钮</p-button>
  </div>
</template>
```

## 组件列表

- pVerificationCode 验证码
