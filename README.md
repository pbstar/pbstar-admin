# Pbstar-admin

## 简介

Pbstar-admin 是一套后台管理系统解决方案。

## 快速开始

```bash
# 克隆项目
git clone https://github.com/pbstar/pbstar-admin.git
# 进入项目目录
cd pbstar-admin
# 安装依赖
npm install
# 创建业务包
npm run create
# 启动主包
npm run dev:main
```

## 项目结构

```
├── cli              # 脚手架
├── components       # 组件
├── main             # 主包
├── packages         # 业务包
├── .gitignore       # git忽略
├── package.json     # 依赖
└── README.md        # 说明
```
